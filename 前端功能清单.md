## 二、功能模块详情

### 2.1 首页功能

#### 2.1.1 顶部公告条
- 显示促销文案（新用户注册即送 888 体验金 | 首存100%红利）
- 金色文字配深色渐变背景
- 火焰图标闪烁动画
- **后台配置**：公告文案可在后台配置

#### 2.1.2 Banner轮播
- 支持多张Banner图片轮播（当前3张）
- 自动播放（4秒间隔），支持手动点击圆点切换
- 包含两个CTA按钮：
  - 「立即试玩」- 平滑滚动到游戏区域
  - 「了解更多」- 跳转到优惠活动页
- **后台配置**：Banner图片、标题、跳转链接

#### 2.1.3 游戏分类
- 6个分类入口：真人、棋牌、电子、体育、彩票、捕鱼
- 点击分类筛选对应游戏并更新列表标题
- 显示筛选结果数量反馈弹窗
- **后台配置**：分类图标、名称、排序、状态

#### 2.1.4 中奖滚动公告
- 横向滚动显示中奖信息
- 无限循环滚动动画
- **后台配置**：中奖信息内容

#### 2.1.5 热门游戏列表
- 4列网格布局（PC端）/ 2列（移动端）
- 显示游戏封面、名称、厂商
- 支持HOT/NEW/VIP标签
- 点击游戏显示进入提示
- **后台配置**：游戏列表、排序、标签、状态

### 2.2 优惠活动页

#### 2.2.1 活动列表
- 卡片式布局展示活动
- 显示活动封面、标签、标题
- 点击查看活动详情
- **后台配置**：活动图片、标题、标签、时间、内容

### 2.3 朋友圈/动态页

#### 2.3.1 专属提示条
- 显示注册引导文案（🔔 注册即可发布动态、获得官方打赏）
- 金色渐变背景（#ffd700 → #ffaa00）
- **后台配置**：提示文案可在后台配置

#### 2.3.2 顶部Banner图片
- 显示朋友圈专属Banner图片
- 图片自适应宽度
- **后台配置**：Banner图片URL

#### 2.3.3 Tab切换
- 三个Tab：官方动态、大舞台、公告中心
- 点击切换不同内容

#### 2.3.4 动态列表
- 显示发布者头像、昵称、时间
- 显示动态内容
- 显示点赞数
- **后台配置**：动态内容、发布者、时间

## 三、交互功能

### 3.1 按钮交互反馈
| 操作 | 生产环境行为 |
|------|-------------|
| 点击「立即试玩」 | 平滑滚动到游戏列表区域 |
| 点击「了解更多」 | 跳转到优惠活动页 |
| 点击游戏卡片 | 调用游戏接口，进入对应游戏试玩 |
| 点击分类图标 | 筛选游戏列表，更新标题为对应分类名称 |
| 点击活动 | 跳转到活动详情页面 |
| 点击登录 | 跳转外部登录URL（从后台配置获取） |
| 点击注册 | 跳转外部注册URL（从后台配置获取） |
| 点击Telegram | 直接跳转Telegram链接（从后台配置获取） |
| 点击WhatsApp | 直接跳转WhatsApp链接（从后台配置获取） |

### 3.2 弹窗系统
- 反馈弹窗：图标、标题、描述、确认按钮（仅Demo演示用，生产环境按实际业务处理）

## 四、响应式设计

| PC端 | >768px | Web适配 |

| 元素 | PC端 | 移动端 |
|------|------|--------|
| 导航菜单 | 显示（首页、优惠、朋友圈、真人、棋牌、电子、体育、彩票、捕鱼） | 隐藏 |
| 底部导航 | 隐藏 | 显示（首页、优惠、朋友圈） |

| Banner标题 | 48px | 20px |
| Banner圆角 | 12px | 8px |
| 游戏网格 | 4列 | 2列 |
| 游戏封面高度 | 150px | 100px |
| 分类网格 | 6列 | 3列 |
| 分类图标尺寸 | 70px | 50px |
| 登录按钮 | 显示 | 显示（适配尺寸 8px 16px） |
| 注册按钮 | 显示 | 显示（适配尺寸 8px 16px） |
| 悬浮按钮尺寸 | 50px | 42px |
| 悬浮按钮位置 | bottom: 100px, right: 20px | bottom: 80px, right: 10px |
