一、功能概述

1.1 功能定位

朋友圈是试玩网的社交互动模块，类似微信朋友圈，用于展示官方公告、用户晒单分享，增强用户粘性和平台活跃度。

1.2 核心价值

  • 官方宣传：发布活动公告、优惠信息，触达用户
  • 用户互动：用户分享中奖截图，形成社区氛围
  • 信任背书：真实用户晒单增强新用户信任感
  • 活跃留存：社交功能提升用户停留时间和回访率

1.3 页面入口

端入口位置
PC端顶部导航菜单「朋友圈」
H5端底部Tab导航「朋友圈」

二、页面结构

2.1 页面布局

区域说明是否可配置
顶部Banner朋友圈专属背景图，下滑自动隐藏后台配置
导航栏返回按钮、标题、发布按钮固定
Tab切换官方动态 / 大舞台固定
动态列表动态卡片流式布局内容可配置
评论输入框点击评论时从底部弹出固定

2.2 Tab分类说明

Tab名称内容类型发布权限
官方动态平台公告、活动通知、官方回复仅管理员
大舞台用户晒单、中奖分享、心得交流注册用户

三、动态卡片结构

3.1 卡片组成元素

元素说明必填
用户头像圆角方形，44×44px必填
用户名显示昵称必填
用户徽章官方标识 或 登录天数可选
个性签名用户自定义签名可选
动态内容文字内容，支持展开收起可选
图片/视频最多9张图片或1个视频可选
发布时间相对时间或绝对时间必填
点赞按钮点赞数量和点赞用户列表必填
评论按钮点击弹出底部评论框必填

3.2 图片网格布局

图片数量布局方式
1张单图展示，最大宽度240px
2张2列网格
3张3列网格
4张2×2网格
5-6张3列网格
7-9张3×3网格

四、用户等级系统

4.1 等级规则

核心规则：登录1天 = 1级，累计登录天数即为用户等级
字段说明示例
登录天数用户累计登录平台的天数47天
徽章显示红色背景徽章，显示"XX天"badge-days

4.2 徽章类型

徽章类型适用对象样式
官方徽章管理员发布的动态紫色渐变背景，显示"官方"
天数徽章普通用户动态红色渐变背景，显示"XX天"

五、评论功能

5.1 评论交互流程

  1. 用户点击动态卡片的「评论」按钮
  2. 底部弹出评论输入框（带遮罩层）
  3. 输入框自动获取焦点
  4. 用户可输入文字，也可添加图片（最多3张）
  5. 点击「发送」提交评论
  6. 评论成功后关闭输入框，刷新评论列表

5.2 评论输入框功能

功能说明
文字输入支持多行文字输入
图片上传点击「+」添加图片，最多3张
图片预览已添加的图片显示缩略图
删除图片点击图片右上角「×」删除
发送按钮提交评论内容

5.3 评论显示规则

  • 默认显示前4条评论
  • 超过4条显示「查看更多」
  • 官方回复显示「官方」标识
  • 支持回复他人评论

六、发布功能

6.1 发布入口

位置说明显示条件
导航栏右侧「+」按钮，紫色渐变背景始终显示

6.2 发布弹窗内容

元素说明
取消按钮关闭发布弹窗
标题"发布动态"
发表按钮提交动态，内容为空时禁用
文字输入框placeholder: "这一刻的想法..."
媒体预览区显示已上传的图片/视频
图片按钮选择图片，最多9张
视频按钮选择视频，仅1个（与图片互斥）

6.3 发布权限

权限说明：
  • 未登录用户：点击发布按钮弹出登录提示
  • 已登录用户：可在「大舞台」Tab发布动态
  • 管理员：可在「官方动态」Tab发布内容

七、后台配置项

关联后台：admin-all-in-one.html > 内容管理 > 朋友圈动态

7.1 动态管理

配置项说明操作
官方动态列表查看所有官方发布的动态查看、编辑、删除
发布官方动态管理员发布官方公告新增、编辑、删除
审核设置用户动态是否需要审核开启/关闭

7.2 大舞台帖子管理

功能说明：管理用户在「大舞台」发布的所有帖子，支持筛选、审核、隐藏、删除等操作

7.2.1 筛选条件

筛选项说明
用户名按用户昵称模糊搜索
登录天数1-30天 / 31-60天 / 60天以上
状态待审核 / 已发布 / 已隐藏
时间范围按发布时间筛选

7.2.2 批量操作

操作说明
批量通过将选中的待审核帖子设为已发布
批量隐藏将选中的帖子设为隐藏状态
批量删除永久删除选中的帖子

7.2.3 帖子详情

信息项说明
用户信息头像、昵称、登录天数、个性签名
帖子内容文字内容全文
媒体内容图片列表或视频预览
互动数据点赞数、评论数、点赞用户列表
评论列表所有评论内容及评论者信息

7.3 Banner配置

配置项说明建议值
Banner图片URL朋友圈顶部背景图尺寸：750×300px
是否显示控制Banner显示/隐藏默认显示

7.4 评论管理

配置项说明
评论列表查看所有评论，支持删除敏感内容
敏感词过滤配置敏感词库，自动过滤
图片评论是否允许用户发送图片评论

八、数据字段说明

8.1 动态数据结构

字段名类型说明
idNumber动态唯一标识
userString用户昵称
avatarString头像URL或emoji
badgeString徽章类型：official/days
daysNumber登录天数（badge为days时）
signatureString个性签名
contentString动态文字内容
imagesArray图片列表（最多9张）
videoString视频URL（与图片互斥）
timeString发布时间
likesArray点赞用户列表
likeCountNumber点赞总数
commentsArray评论列表

8.2 评论数据结构

字段名类型说明
userString评论用户昵称
textString评论文字内容
imagesArray评论图片（最多3张）
isOfficialBoolean是否为官方回复
replyToString回复的用户（可选）
