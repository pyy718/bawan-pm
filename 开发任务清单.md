## 一、前端开发（H5/Web）

### 1. 首页模块

#### 1.1 顶部公告条
- 显示促销文案（金色文字 + 深色渐变背景）
- 火焰图标闪烁动画
- 文案内容从后台配置获取

#### 1.2 头部导航
- Logo展示（图标 + 网站名称）
- PC端：9个导航菜单（首页、优惠、朋友圈、真人、棋牌、电子、体育、彩票、捕鱼）
- 登录/注册按钮（跳转外部URL，从后台配置获取）
- 毛玻璃效果 + 吸顶固定

#### 1.3 Banner轮播
- 支持多张轮播（4秒自动切换）
- 圆点指示器 + 手动切换
- 两个CTA按钮：「立即试玩」滚动到游戏区、「了解更多」跳转优惠页
- Banner图片、标题、按钮链接从后台配置

#### 1.4 游戏分类
- 6个分类入口（真人、棋牌、电子、体育、彩票、捕鱼）
- 点击筛选对应游戏 + 更新列表标题
- 分类图标、名称、颜色从后台配置

#### 1.5 中奖滚动公告
- 横向无限循环滚动
- 显示中奖信息（玩家、游戏、金额）
- 内容从后台配置

#### 1.6 热门游戏列表
- 4列网格（PC）/ 2列网格（H5）
- 游戏卡片：封面、名称、厂商、标签（HOT/NEW/VIP）
- 点击调用游戏接口进入试玩
- 游戏数据从后台接口获取

### 2. 优惠活动页

#### 2.1 活动列表
- 卡片式布局展示
- 显示：封面图、标签、标题、有效期
- 点击跳转活动详情页
- 活动数据从后台接口获取

### 3. 朋友圈页

#### 3.1 专属提示条
- 金色渐变背景提示条
- 显示：「🔔 注册即可发布动态、获得官方打赏」

#### 3.2 顶部Banner图片
- 图片自适应宽度、无间距
- 图片URL从后台配置

#### 3.3 Tab切换
- 三个Tab：官方动态、大舞台、公告中心
- 点击切换不同内容列表

#### 3.4 动态列表
- 显示：头像、昵称、时间、内容、点赞数
- 点击跳转动态详情页
- 动态数据从后台接口获取

### 4. 公共组件

#### 4.1 悬浮联系按钮
- 固定右下角（PC: bottom 100px / H5: bottom 80px）
- Telegram + WhatsApp 两个按钮
- 点击直接跳转对应链接（链接从后台配置）

#### 4.2 弹窗系统
- 反馈弹窗：图标、标题、描述、确认按钮（仅Demo演示用）

#### 4.3 底部导航（H5专用）
- 三Tab导航：首页、优惠、朋友圈
- 毛玻璃效果 + 固定底部
- 当前页高亮显示

### 5. 响应式适配（示例）

| 元素 | PC端（>768px） | H5端（≤768px） |
|------|----------------|----------------|
| 顶部导航菜单 | 显示9个菜单 | 隐藏 |
| 底部Tab导航 | 隐藏 | 显示3个Tab |
| 登录按钮 | 显示 | 显示（适配尺寸） |
| Banner高度 | 400px | 220px |
| Banner标题 | 48px | 20px |
| 游戏网格 | 4列 | 2列 |
| 分类网格 | 6列 | 3列 |
| 悬浮按钮 | 50px | 42px |

## 二、后台开发（Admin）

### 1. 登录模块
- IP白名单验证（只有白名单IP可登录）
- Google验证码二次验证
- 角色权限控制（超级管理员/运营/客服）
- 登录日志记录

### 2. 数据统计（首页仪表盘）

#### 2.1 统计卡片
- 今日点击量（实时）
- 今日试玩人数
- 今日注册人数
- 总注册用户数

#### 2.2 趋势图表
- 7日/30日数据趋势折线图
- 24小时分布柱状图

#### 2.3 排行榜
- 游戏点击排行TOP10
- 注册来源排行

### 3. 用户管理

#### 3.1 后台用户
- 用户列表（账号、角色、状态、最后登录）
- 新增/编辑/删除用户
- Google密钥管理（生成/重置）
- 角色分配

#### 3.2 试玩用户
- 试玩记录列表
- 筛选：时间、游戏、来源

#### 3.3 注册用户
- 注册用户列表
- 筛选：时间、来源渠道
- 导出功能

### 4. 游戏管理

#### 4.1 游戏类别
- 类别列表（真人、棋牌、电子、体育、彩票、捕鱼）
- 新增/编辑/删除类别
- 排序、状态管理

#### 4.2 游戏列表
- 游戏CRUD（封面、名称、类别、厂商、标签）
- 批量导入/导出
- 排序、状态管理

#### 4.3 已删除游戏
- 回收站功能
- 恢复/彻底删除

#### 4.4 推荐游戏
- 多主题推荐配置
- 拖拽排序

### 5. 内容管理

#### 5.1 Banner管理
- Banner列表（排序、预览、标题、链接、位置、状态）
- 新增/编辑/删除Banner
- 图片上传（建议1200x400px）
- 按钮跳转配置（立即试玩URL、了解更多URL）

#### 5.2 优惠活动
- 活动列表（封面、标题、标签、时间、状态）
- 新增/编辑/删除活动
- 标签管理（新人首存/会员专享/限时活动）

#### 5.3 朋友圈动态
- Tab分类管理（官方动态/大舞台/公告中心）
- 动态列表（发布者、内容、点赞数、时间、状态）
- 发布/编辑/删除动态

### 6. 下游配置

#### 6.1 网页配置
- Web端登录URL（含推广参数）
- H5端登录URL（含推广参数）
- Web端注册URL（含推广参数）
- H5端注册URL（含推广参数）

#### 6.2 APP配置
- iOS描述文件地址
- Android下载链接

#### 6.3 联系方式
- 联系方式列表（类型、名称、链接、状态）
- 支持：Telegram/WhatsApp/Line/Zalo/其他
- 新增/编辑/删除

### 7. 系统管理

#### 7.1 网站配置
- 网站名称、副标题
- Logo图片上传
- 版权信息

#### 7.2 站内信
- 消息列表（标题、接收对象、时间、状态）
- 发送通知（全部用户/注册用户/指定用户）

#### 7.3 IP白名单
- 白名单列表（IP、备注、添加时间）
- 新增/编辑/删除IP
- 支持CIDR格式（如10.0.0.0/24）

## 三、接口对接

### 1. 前端接口（H5/Web调用）

| 接口 | 方法 | 说明 |
|------|------|------|
| /api/config/site | GET | 获取网站配置（名称、Logo等） |
| /api/config/contact | GET | 获取联系方式列表 |
| /api/config/login-url | GET | 获取登录跳转URL |
| /api/config/register-url | GET | 获取注册跳转URL |
| /api/banner/list | GET | 获取Banner列表 |
| /api/category/list | GET | 获取游戏分类列表 |
| /api/game/list | GET | 获取游戏列表（支持分类筛选） |
| /api/game/hot | GET | 获取热门推荐游戏 |
| /api/promo/list | GET | 获取优惠活动列表 |
| /api/moments/list | GET | 获取朋友圈动态列表 |
| /api/marquee/list | GET | 获取中奖滚动公告 |

| 游戏 | /admin/category/* | 游戏类别CRUD |
| 游戏 | /admin/game/* | 游戏列表CRUD |
| 内容 | /admin/banner/* | Banner管理CRUD |
| 内容 | /admin/promo/* | 优惠活动CRUD |
| 内容 | /admin/moments/* | 朋友圈动态CRUD |
| 配置 | /admin/config/* | 系统配置管理 |
| 配置 | /admin/whitelist/* | IP白名单管理 |

## 六、技术栈建议

### 前端（H5/Web）
- HTML5 + CSS3 + JavaScript
- 响应式布局（Flexbox + Grid）
- 可选框架：Vue3 / React

### 后台（Admin）
- Vue3 + Element Plus
- ECharts（图表）
- Axios（HTTP请求）

### 后端
- Node.js / Java / Go（按团队技术栈选择）
- MySQL / PostgreSQL
- Redis（缓存）
