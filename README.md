# 试玩网 - 游戏试玩推广平台

## 项目说明

这是一个游戏试玩推广平台的原型演示项目，包含前端展示页面和后台管理系统。

## 在线访问

- **前端页面**: `/` 或 `/frontend-demo.html`
- **后台管理**: `/admin` 或 `/admin-all-in-one.html`

## 后台登录信息

- 账号: `admin`
- 密码: `123456`
- Google验证码: 留空即可

## 技术栈

- HTML5 + CSS3 + JavaScript
- ECharts (图表库)
- 响应式设计 (支持PC和移动端)

## 本地开发

直接用浏览器打开 HTML 文件即可预览。

## 部署

本项目已配置 Vercel 部署，推送到 GitHub 后可自动部署。
